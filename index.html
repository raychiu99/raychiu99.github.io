InTrade

   

I developed a stock chatbot assistant which allows users to get up to date stock prices. User functionalities include calculating the EMA's, SMA's, and RSI's of companies. The chat bot's interface, built by StreamLit, allows users to view historical stock performance through plot visualizations. I integrated OpenAI's API using python and the yfinance library to provide up to date stock prices.

Check it out!
